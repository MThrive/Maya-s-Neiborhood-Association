A Note on Bot Protection

This page is built to reduce automated scraping of contact information:

Contact details stay hidden in the page markup and are only assembled and displayed by JavaScript after a real hover, tap, or keyboard focus event — this defeats most simple scrapers that just read static HTML. See SECURITY-RECOMMENDATIONS.md (included with this template) for additional server-level and hosting-level precautions, since a fully bot-proof, geographically-restricted site requires configuration beyond what plain HTML/CSS/JS can enforce.
